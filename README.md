## Projects

### Scaffold

1. Go to scaffold directory
    ```bash
    cd scaffold
    ```
2. Create Python virtual environment
    ```bash
    python3 -m venv ~/.scaffold
    ```
3. Activate the venv
    ```bash
    source ~/.scaffold/bin/activate
    ```
4. Install required packages
    ```bash
    make install
    ```

    Useful commands
    
    ```bash
    make all # install run test at once
    deactivate # exit the venv
    ```
